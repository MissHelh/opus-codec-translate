Opus

Opus编解码器被设计为基于网络的交互语音和音频传输。由IETF工作组结合Skype的SILK和Xiph.Org的CELT解码器设计而成。

Opus编解码器被设计用于处理各种交互式音频应用，包括IP语音，视频会议，游戏内聊天，远程的现场音乐表演等。处理范围从低比特率窄带语音到高质量立体声音乐。其主要特性有：

  • 8 kHz ~ 48 kHz 采样率
  • 6 kb/s ~ 510 kb/s 比特率
  • 支持静态码率(CBR)和动态码率(VBR)
  • 音频带宽从窄带到全频带
  • 支持语音和音乐
  • 支持单声道和立体声
  • 支持多频道(最多255频道)
  • 动态可调比特率、音频带宽、帧大小
  • 良好的鲁棒性和数据包丢失隐藏(PLC)
  • 浮点和定点实现
